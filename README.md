# PrivacyPal Phase 2 MVP

This is the Phase 2 MVP version of the PrivacyPal client-side quiz tool.

## Contents:
- index.html — Main quiz interface
- style.css — Basic mobile-first styling
- logic.js — Quiz logic (updated scoring)
- assets/ — Icons and shield graphics
- .nojekyll — Ensures proper GitHub Pages rendering

## Hosting Notes:
To deploy:
1. Upload contents to GitHub repository
2. Enable GitHub Pages (root directory)
3. Or embed via iframe into WordPress, Notion, etc.
